AI Crowd Panic Detection System

PanicVision is an Artificial Intelligence-based Crowd Panic Detection System designed to monitor crowded public places and identify abnormal crowd density conditions using computer vision techniques.

🎯 Project Objective

The main objective of this system is to detect crowd density levels and predict possible panic situations using image and video analysis. It helps authorities take early action before emergencies occur.

🧠 Technologies Used

  • Python Programming
  • Flask Web Framework
  • OpenCV (Computer Vision)
  • YOLOv8 Object Detection AI Model
  • HTML, CSS (Frontend Design)
  • Backend Processing System
  • Database SQLite
  • Telegram Bot API (Real-time Alerts)
  • Local Siren System (Audio Alert)

🚀 Key Features

  • User Registration and Login System
  • Image-based Crowd Detection
  • Video-based Crowd Detection
  • AI-based Human Detection using YOLO
  • Real-time People Counting
  • Crowd Classification (Low, Medium, High, Panic)
  • Automatic Panic Alert System
  • Siren Alert for Emergency Situations
  • Telegram Notification Alerts
  • Dashboard with Analytics
  • Log Storage and History Tracking
  • Crowd Detection Trend Graph

🌍 Applications

  • Public Transportation Hubs
  • Events & Festivals
  • Educational Institutions
  • Smart Cities & Public Places
  • Emergency & Disaster Management
  • Religious Places
  • Airports & Security Zones
  • Sports Stadiums

📊Project Outcome

This system successfully detects the number of people in crowded environments and classifies crowd density levels into Low, Medium, High, and Panic Alert conditions using Artificial Intelligence. Classifies crowd levels into Low, Medium, High, and Panic Alert for real-time safety monitoring.
